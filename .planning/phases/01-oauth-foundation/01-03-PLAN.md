<task type="auto">
  <name>Add static file serving and session status API</name>
  <files>server.js</files>
  <action>
Update server.js to add:

**Static file serving:**
```javascript
app.use(express.static('public'));
```
Place this AFTER session middleware but BEFORE route handlers.

**Session status API endpoint:**
```javascript
app.get('/api/session-status', (req, res) => {
  const userA = req.session.tokens?.userA ? {
    displayName: req.session.tokens.userA.displayName,
    spotifyId: req.session.tokens.userA.spotifyId
  } : null;
  
  const userB = req.session.tokens?.userB ? {
    displayName: req.session.tokens.userB.displayName,
    spotifyId: req.session.tokens.userB.spotifyId
  } : null;
  
  res.json({ userA, userB });
});
```

**Update callback redirect in routes/auth.js:**
- Success: `res.redirect('/?status=success')`
- Error: `res.redirect('/?error=' + encodeURIComponent(errorMessage))`

**Add session clear endpoint (for testing/restart):**
```javascript
app.post('/api/clear-session', (req, res) => {
  req.session.destroy((err) => {
    if (err) return res.status(500).json({ error: 'Failed to clear session' });
    res.json({ success: true });
  });
});
```
